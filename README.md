# bingleshop-backend
repository for the API Using ExpressJS and PostgresSQL #challengeGoldBinarAcademy

First, run the development server:
```
npm install
npm run start:dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
